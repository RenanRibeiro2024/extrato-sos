# O que é o Git
É um software que nos ajuda a fazer versionamento de projetos
## Como Utilizar
Entre no site : https://git-scm.com/downloads e faça o download e a instalação
Depois de instalar o Git, os comandos : git init para iniciar um projeto

## Área de trabalho, área de preparação e repositórios no git
No git Trabalhamos com etapas de projeto. A etapa inicial é o que alteramos no nosso computador sem controle de versão que é a área de trabalho.
