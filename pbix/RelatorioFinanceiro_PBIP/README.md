# RelatorioFinanceiro_PBIP (fictício — completo)
Projeto PBIP de exemplo com tabelas fictícias: Customers, Products, Sales.

Como usar:
1. Extraia a pasta para seu disco.
2. Coloque os arquivos CSV (Customers.csv, Products.csv, Sales.csv) na mesma pasta do project.json ou ajuste as consultas M.
3. Abra o Power BI Desktop -> File -> Open -> Project (.pbip) e selecione project.json.
4. Caso necessário, em Transform Data -> Source Settings, ajuste as permissões e caminhos locais.
